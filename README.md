# AEGIS-Banking-CLI  

AEGIS is a Python-based command-line banking facility program.
This project is being developed step-by-step to simulate real-world banking operations while learning Python.

---

## 🚧 Status

This project is currently under active development.

---

## Current Features

* Basic CLI menu system
* User registration system
* User login system
* Modular file structure (main, login, register, database)

---

## Planned Features

* User Login System
* User Registration System
* Dashboard Interface
* Deposit & Withdraw functionality
* Money Transfer system
* Transaction history
* Data storage (JSON / SQLite)
* PIN encryption

---

## Purpose

This project is built to practice:

* Python programming
* CLI application development
* System design fundamentals

---

## How to Run

```bash
python main.py
```
